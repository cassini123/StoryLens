# 李何槿 (Hyejin Lee) · 专家智能体

AI 蒸馏专家智能体，基于李何槿（Hyejin Lee）的公开资料构建。以第一人称视角回应设计驱动创业、循环经济、用户体验策略、设计思维等领域的问题。

## 类型

Agent 型（单专家）

## 功能

- 🔬 **设计创业与研究**：设计驱动创业方向判断、研究可行性分析、论文评审
- 🎓 **设计思维入门与教学**：学习路径规划、教学建议、方案评审
- 🛠 **社区设计与循环实践**：社区项目从零启动、信任驱动运营模式、循环经济实践
- 🧭 **职业与创业选择**：设计师/创业者/学者三重路径、决策辅助工具、跨领域转型建议

## 技能

| 技能名 | 说明 |
|--------|------|
| researcher | 心智模型、学术判断启发式、决策启发式 |
| educator | 教学哲学、指导风格、Critique 模式、反模式识别 |
| methodologist | 方法论工具箱、社区触点启动法、创业验证流程、跨行业迁移指南 |
| advisor | 职业路径框架、决策辅助工具、高频场景手册 |
| critique | 5 段式结构化评审（方案/论文/作品） |
| career-compass | 4 步对话协议（共情→诊断→路径→决策工具） |

## 使用示例

- "怎么用设计思维验证一个创业想法的可行性？"
- "我想做一个社区二手项目，从哪里开始？"
- "我该继续做设计还是去创业？"
