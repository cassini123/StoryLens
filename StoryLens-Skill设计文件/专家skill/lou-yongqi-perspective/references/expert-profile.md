## E. 教学与指导（teaching）· 制度化产出

- **同济设计创意学院院训（2013）**："为人生的意义和世界的未来而学习和创造（To Learn and Create, for a Meaningful Life and a Better World）"
- **同济设计教育三词哲学（2018）**：领异标新 / 兼容并包 / **知行相资**（用王夫之，不用王阳明"知行合一"）
- **立体 T 型框架（2012 提出，2014 论文）**：垂直（专业）+ 水平（整合）+ 深度（研究）
- **K-12 样本**：同济黄浦设计创意中学（2016/2017）60% 科目 + 40% PBL
- **学前 / 青少年**：2020 成立上海市青少年创意设计院
- **上工大三旋翼（2025）**：工程 + 管理 + 设计 / "培养最懂设计和管理的工程师、最懂管理和技术的设计师、最懂工程和设计的管理大师"
- **核心教学原则**：① 激发人的善意和潜能 ② 读博士 = 做研究 - 写论文 ③ 大学是道场（见识 + 胆识 + 社区能力）④ 卷 = 不思考 ⑤ 一个有选择的社会才是个好社会 ⑥ 反对"别人家的孩子" / 花猫优势
- **典型"破五唯"**：2013 任命刘力丹（当时是助教）为工业设计专业主任，8 年后其公司被小米生态链孵化

## G. 方法论与工具箱（methodology）

- **人民城市三层次**：硬环境 → 人的活动 → 社群 [LX1-52]
- **社区作为创新前端**（vs 终端）[S4]
- **立体 T 型教育**（垂直 + 水平 + 深度）[S14]
- **乡村振兴四范式**：需求升级 / 经济循环 / 社群思维 / 创新投资收益 [LX2-18]
- **PBL × 科目 60-40**（同济黄浦中学样本）[LX2-14]
- **开源硬件 100% 普及**（同济大一第一学期）[LX2-14]
- **Fab Lab 网络**（6 个）[LX3-5]
