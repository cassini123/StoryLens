**代表著作**：

## G. 方法论与工具箱（methodology）

**核心理论框架**（★ 私域全文验证）：

| 框架 | 核心要素 | 私域来源 |
|------|---------|---------|
| **交互五要素** | People / Purpose / Contexts / Means / Actions | 公域 |
| **行为逻辑** | 物理逻辑→行为逻辑的认识论转向 | 公域 + S-pri-002 |
| **UX→EX 范式** | 体验从评价准则变为设计对象；Creator→Enabler | S-pri-003 全文 |
| **IDR 方法** | Initiation → Development → Recognition 三阶段 | S-pri-001 全文 |
| **BEV-E 模型** | Behavior / Environment / Value / Experience 四维 | S-pri-001 全文 |
| **A-C/E-M 框架** | 附加价值-核心主体 × 效率-意义 四象限 | S-pri-005 全文 |
| **服务设计五要素** | 基于戏剧"五位一体"理论 | S-pri-008 全文 |
| **心理草绘** | 在先意向与即时意向的动态交互 | S-pri-029 |

**框架间内在关联**：
```
交互五要素（微观：行为要素）
  → 行为逻辑（中观：行为组织）
    → 体验设计范式（中观→宏观：体验作为对象）
      → 服务设计定位（宏观：组织战略，A-C/E-M）
        → IDR 生活方式设计（社会系统：蝴蝶效应，BEV-E）
          → 设计哲学实践（元层面：学术传承与创新）
            → 从设计师到公民（终极命题：设计的社会责任）
```

**研究方法特征**：概念定义先行、跨尺度串联、范式对比、认识论追问

## H. 价值观与立场（values）

**追求的核心价值**：
1. **行为作为设计的核心对象** — 设计的本质是组织行为
2. **设计意义的层级跃升** — 从"生活的手段"到"生命的意义"
3. **设计的社会维度** — 从设计师到公民
4. **创造知识而非传递知识** — 设计教育的终极目标
5. **设计哲学作为根基** — 布坎南学术传统的继承与发展
6. **跨学科整合** — 不同知识系统的交叉是创新的源泉

**不认同的做法**：把交互设计等同于 UI 设计；把 UX 简化为满意度指标；理论与实践的割裂；学科壁垒；设计师社会责任被忽略

**自己也在思考的**：AI 对交互设计的影响；IDR 方法的规模化；设计哲学的实践化；从设计师到公民的路径；心理草绘的方法论深化

**领域使命感**：在中国交互设计从 0 到 1 的关键时期，同时承担了学科建设者、理论奠基人、范式推动者三重角色
