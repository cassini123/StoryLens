# 分镜视觉语言手册

## 镜头类型与情感功能

### 景别 (Shot Size) — 控制信息量与亲密感

| 景别 | 画面范围 | 情感功能 | 使用时机 |
|------|---------|---------|---------|
| **远景/全景** | 全场景+环境 | 建立空间感、渺小感或壮丽感 | 开场定调、转场、展示环境 |
| **中景** | 膝盖以上 | 展示人物关系、互动、动作 | 对话、群体场景 |
| **近景** | 胸部以上 | 传达情绪、表情细节 | 关键情绪转折、内心戏 |
| **特写** | 面部局部 | 极致聚焦、压迫感或亲密感 | 高潮点、关键道具、眼神交流 |
| **极特写** | 单一细节（眼睛/手） | 象征性、隐喻性表达 | 转折暗示、伏笔回收 |

**节奏原则**：
- 连续同景别 ≤ 3 帧，否则视觉疲劳
- 大景别切换到小景别 = "推进"（紧张/聚焦）
- 小景别切换到大景别 = "拉开"（释放/揭示）

### 角度 (Camera Angle) — 暗示权力关系

| 角度 | 效果 | 心理暗示 |
|------|------|---------|
| **平视** | 客观、平等 | 正常叙事视角 |
| **俯拍** | 渺小、脆弱、被观察 | 弱势角色、展示孤立感 |
| **仰拍** | 强大、威严、威胁 | 反派登场、英雄时刻 |
| **荷兰角** | 不安、失衡、异常 | 精神不稳定、超自然事件 |

### 运镜 (Camera Movement) — 引导注意力

- **固定镜头**：稳定、正式、观察者视角
- **推 (Dolly In)**：强调、逼近心理、增加紧张
- **拉 (Dolly Out)**：揭示、孤独感、"被抛弃"的感觉
- **摇 (Pan)**：跟随动作、展示环境关联
- **跟 (Tracking)**：沉浸感、与角色同行
- **环绕 (Arc)**：展示360°关系、戏剧性揭示

## 分镜编排的节奏曲线

### 标准三幕节奏

```
张力 ↑
    │
高 │         ╱╲        ╱╲
    │        ╱  ╲      ╱  ╲
中 │   ╭───╯    ╰────╯    ╰────
    │  ╱                        ╲
低 │ ╱                          ╰────→
    ├────────┬─────────┬──────────→ 时间
    │  第一幕  │  第二幕  │  第三幕
    │  (建置)  │  (对抗)  │  (解决)
```

### 分镜层面的微观节奏

每组分镜（6-12帧）内部应有自己的 mini 节奏：

1. **开场帧**：建立（地点 + 人物 + 基调）
2. **发展帧**：推进（动作/对话/信息递进）
3. **高潮帧**：转折（意外/冲突/发现）
4. **收尾帧**：落地（反应/余韵/钩子）

### 情绪温度计

给每帧标注情绪值（-5 到 +5），检查序列：

```
示例：一场争吵戏
Frame 1:  +1 （平静日常）
Frame 2:  +2 （微小不快）
Frame 3:  0  （沉默对峙）
Frame 4: -2 （首次爆发）
Frame 5: -4 （激烈冲突）
Frame 6: -3 （余怒未消）
Frame 7: -1 （开始冷静）
Frame 8:  0  （表面平静，暗流涌动）
```

**检查项**：是否有单调段（连续5帧情绪值变化 < 1）？是否有过山车式跳跃缺乏过渡？

## 转场语法

### 基础转场

| 类型 | 用法 | 效果 |
|------|------|------|
| **切 (Cut)** | 最常用 | 直接、干净、节奏快 |
| **淡入/淡出** | 场景转换 | 时间流逝、梦幻感 |
| **叠化** | 联想/对比 | 两画面的意义叠加 |
| **划像** | 地点切换 | 明确的空间转移 |

### 语义化转场

高级分镜利用转场本身传递信息：

- **匹配切 (Match Cut)**：形状/颜色/动作相似的两帧连接 → 暗示深层联系
- **跳切 (Jump Cut)**：同一主体不同瞬间的跳接 → 时间压缩/不安感
- **L-Cut / J-Cut**：声音先于画面（L）或画面先于声音（J） → 自然流畅的时空过渡
- **视线匹配 (Eyeline Match)**：角色看向某处 → 切到其所看之物 → 建立"共享现实"

## 180度规则与空间连贯性

### 180度线

在双人对话场景中，想象一条穿过两人的直线。摄像机只能在这条线的**一侧**拍摄。

```
     [A] ←——————— 180度线 ———————→ [B]
           ╱ 拍摄区 ════╲ 拍摄区
          ╱                    ╲
```

违反此规则的后果：观众会感觉 A 和 B 的位置关系突然反转，产生空间混乱。

### 例外情况（故意打破规则时）

- 穿越轴线可以表现"关系逆转"或"世界颠倒"
- 但必须让观众明确意识到这是有意为之

## 动态分镜的特殊考量

### 游戏 vs 影视分镜的差异

| 维度 | 影视分镜 | 游戏（动态）分镜 |
|------|---------|------------------|
| **时间控制** | 导演精确控制 | 玩家行为不可预测 |
| **镜头语言** | 固定序列 | 需要设计多套应对方案 |
| **交互触发** | 无 | 必须设计触发条件 |
| **重玩价值** | 一次性体验 | 同一场景多次进入 |

### 游戏过场动画 (Cutscene) 设计建议

1. **保持短 (< 90秒)**：玩家耐心有限
2. **可跳过选项**：尊重玩家的自主权
3. **游戏内引擎渲染优于预渲染 CG**：避免风格割裂
4. **无缝衔接**：过场结束后立即交还控制权

### VR 分镜特殊规则

- **无"画框"概念**：用户自己决定看哪里
- **引导注意力的方法**：光线引导、运动物体、声音提示
- **避免快速剪辑**：VR 中切换镜头会导致晕动症
- **舒适区原则**：重要内容放在视线下方30°范围内
